SETUP INSTRUCTIONS FOR FAL.AI MCP SERVER
=========================================

1. COPY SERVER.PY CONTENT:
   - Copy the content from the 'server.py' artifact above
   - Save it to: src/fal_mcp_server/server.py

2. INSTALL DEPENDENCIES:
   cd ~/fal-mcp-server
   uv pip install mcp fal-client
   # OR if you don't have uv:
   pip install mcp fal-client

3. GET YOUR FAL.AI API KEY:
   - Sign up at https://fal.ai
   - Get your API key from the dashboard

4. CONFIGURE CLAUDE DESKTOP:
   - Find your Claude Desktop config file:
     ~/Library/Application Support/Claude/claude_desktop_config.json

   - Add this configuration (replace YOUR_USERNAME and your-fal-api-key):
   {
     "mcpServers": {
       "fal-ai": {
         "command": "uv",
         "args": [
           "run",
           "--with",
           "mcp",
           "--with",
           "fal-client",
           "python",
           "/Users/YOUR_USERNAME/Desktop/fal-mcp-server/src/fal_mcp_server/server.py"
         ],
         "env": {
           "FAL_KEY": "your-fal-api-key-here"
         }
       }
     }
   }

5. RESTART CLAUDE DESKTOP

6. TEST IT:
   Ask Claude to generate an image!

For questions, check the README.md file.
